# BibleVerseApp
Bible Verse Searcher – ASP.NET Core MVC Web Application

This project is a full-stack Bible Verse Searcher built using ASP.NET Core MVC, demonstrating enterprise-level design principles such as N-Layer architecture, the Data Access Object (DAO) pattern, and Object-Oriented Programming (OOP). The application is structured with clear separation between the Presentation layer, Business Logic, and Data Access Layer, improving maintainability, scalability, and testability.

Users can search for Bible verses by keyword, book, chapter, or verse, making it easy to navigate the 31,000+ verses of Scripture. The application also includes a note-taking feature that allows users to attach personal reflections, sermon notes, or study insights directly to specific verses. Data access is abstracted through interfaces (IBibleVerseDAO, IBibleBookDAO, IVerseNoteDAO), allowing flexible integration with different data sources.

This project demonstrates key concepts in modern web development, including dependency injection, MVC architecture, database abstraction, and clean code organization, while also providing a practical tool for Bible study, teaching, and personal reflection.
